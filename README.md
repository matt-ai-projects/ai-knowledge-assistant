# AI Knowledge Assistant

An enterprise-grade RAG (Retrieval-Augmented Generation) chatbot that answers questions from your uploaded documents.

## What it does

Upload any document — sales playbooks, product manuals, contracts, onboarding guides — and ask questions in plain English. The assistant finds the relevant information and answers accurately, citing only what's in your documents.

## Tech Stack

| Layer | Tool |
|---|---|
| LLM | Amazon Bedrock (Claude) |
| Embeddings | Amazon Titan Embeddings |
| Orchestration | LangChain |
| Vector Database | ChromaDB |
| Frontend | Streamlit |
| Language | Python |

## Getting Started

```bash
git clone https://github.com/YOUR_USERNAME/ai-knowledge-assistant.git
cd ai-knowledge-assistant
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
